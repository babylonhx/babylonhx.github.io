static function MergeMeshes(meshes:Array<Mesh>, ?disposeSource:Bool, ?allow32BitsIndices:Bool, ?meshSubclass:Mesh):Mesh

Merge the array of meshes into a single mesh for performance reasons.

{Array}

meshes - The vertices source. They should all be of the same material. Entries can empty

{boolean}

disposeSource - When true (default), dispose of the vertices from the source meshes

{boolean}

allow32BitsIndices - When the sum of the vertices > 64k, this must be set to true.

{Mesh}

meshSubclass - When set, vertices inserted into this Mesh. Meshes can then be merged into a Mesh sub-class.
